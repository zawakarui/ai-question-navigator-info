🔒Privacy Policy

📋Summary

AI Question Navigator completely protects your privacy:

  • No Data Collection - We do not collect any personal information
  • No External Transmission - All processing happens locally in your browser
  • Minimal Permissions - Only used for question detection and UI display

Last Updated: June 18, 2025
Effective Date: June 18, 2025
Version: 1.0.0

📊Information We Access

Important: This extension does not collect, store, or transmit any personal information.

✅ Information We Access

We temporarily access the following information but never transmit it externally:

  • Web Page Content: Only for detecting user questions on Claude.ai, ChatGPT, and Gemini pages
  • Active Tab Information: Only for determining if the site is supported

❌ Information We Do NOT Collect

  • Personal identifiers (names, email addresses, etc.)
  • Authentication credentials (passwords, login information)
  • Chat history (content other than questions)
  • User behavior patterns
  • Location information
  • Device information

🔑Permissions Used

Permission Purpose Restrictions
activeTab Question detection in current tab Only works on supported sites (Claude, ChatGPT, Gemini)
scripting Display question list UI No page content modification; UI element addition only

💾Data Storage & Management

  • Local Storage: Not used
  • External Servers: No connections
  • Data Retention: Temporary memory storage only while page is displayed
  • Data Deletion: Automatic deletion on page reload

🛡️Security Measures

  • Manifest V3 Compliant: Follows Chrome's latest security standards
  • Principle of Least Privilege: Uses only minimum necessary permissions
  • Sandboxed Execution: Operates within Chrome extension security model
  • Open Source: Ensures code transparency

🌐Third-Party Information Sharing

This extension does not share information with any third parties. No external service integrations, analytics tools, or advertising are used.

👶Children's Privacy Protection

This extension does not collect any personal information regardless of age, ensuring complete privacy protection for children.

🌍International Privacy Law Compliance

  • GDPR (EU): Not applicable as no data collection occurs
  • CCPA (California): We do not sell personal information
  • Personal Information Protection Act (Japan): No personal information handling

🔄Privacy Policy Changes

Important changes to this policy will be announced on the Chrome Web Store extension page. Continued use of the extension after changes constitutes acceptance of the new policy.

📞Contact Us

For privacy-related questions, please contact us through the Chrome Web Store support page. We typically respond within 1-3 business days.

📜Legal Matters

  • Governing Law: Japanese law
  • Jurisdiction: Tokyo District Court

📋Privacy Protection Guarantee

AI Question Navigator is designed with your privacy as the top priority. All processing occurs locally, with no personal information collection or transmission. Use with confidence.
